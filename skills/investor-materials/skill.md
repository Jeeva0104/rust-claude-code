# Investor Materials

## When to Activate
- Fundraising for Rust project
- Demo day preparation
- Investor updates

---

## Pitch Deck Structure

1. **Problem** — What pain point?
2. **Solution** — How Rust solves it (performance, safety)
3. **Market** — TAM/SAM/SOM
4. **Traction** — Downloads, users, revenue
5. **Team** — Rust expertise
6. **Ask** — Funding amount
