# Market Research

## When to Activate
- Evaluating Rust crate ecosystem
- Competitive analysis
- Investor due diligence

---

## Research Framework

| Area | Sources | Metrics |
|------|---------|---------|
| Crate popularity | crates.io, lib.rs | Downloads, dependents |
| GitHub activity | GitHub API | Stars, forks, issues |
| Community | Reddit, Discord | Engagement |
| Alternatives | Awesome Rust lists | Feature comparison |

---

## Competitor: [Name]
