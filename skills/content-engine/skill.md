# Content Engine for Rust

## When to Activate
- Announcing new Rust releases
- Sharing Rust tips
- Building community engagement

---

## Platform Formats

| Platform | Format | Length |
|----------|--------|--------|
| Twitter/X | Thread | 280 chars/tweet |
| Reddit | Code block + explanation | Medium post |
| Discord | Snippet | < 2000 chars |
| Blog | Full tutorial | Long form |

---

## Content Types

### Tip Snippet
```rust
// Did you know? `matches!` macro avoids complex matches
if matches!(result, Ok(42)) { ... }
```

### Before/After
Before: verbose match
After: elegant combinator

## Repurposing Workflow

1. Write blog post
2. Extract 5 tweetable snippets
3. Create Reddit summary
4. Post to relevant Discord channels
